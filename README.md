# Daily Manager App

Ứng dụng quản lý công việc công ty và cá nhân trong ngày.

## Cách dùng nhanh

1. Upload toàn bộ các file/thư mục trong thư mục này lên GitHub.
2. Kết nối GitHub với Vercel.
3. Deploy project trên Vercel.
4. Mở link Vercel bằng Safari trên iPhone.
5. Bấm Share → Add to Home Screen.

## Lưu ý

Bản này lưu dữ liệu trên từng thiết bị bằng localStorage. Chưa có đăng nhập và chưa đồng bộ giữa iPhone/MacBook.
